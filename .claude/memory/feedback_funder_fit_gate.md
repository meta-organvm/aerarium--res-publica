Never optimize a document for a funder whose alignment hasn't been verified against current inventory.

**Why:** Creative Capital 2027 cycle — invested significant drafting energy before discovering the core misalignment: CC funds art audiences encounter sensorially, ORGANVM currently produces governance infrastructure. The Generative Testament (art output) is V2 aspiration, not V1 reality. The rewrite brief was well-crafted but optimizing for a funder that doesn't fit this cycle.

**How to apply:** Before any application drafting begins, run the fit-assessment protocol:
1. What does this funder actually fund? (read winners, not mission statements)
2. What do we actually have right now? (inventory, not aspiration)
3. Is there alignment between 1 and 2? (not "can we frame it" but "does it fit")
4. If no → stop, redirect energy to aligned funder or wait for the right cycle
5. If yes → proceed to drafting

The parry-and-touch process (brief → adversarial review → honest inventory check → go/no-go) is the replicable pattern. Apply to every entry in `applications/`.
