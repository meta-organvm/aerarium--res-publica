# Aspiration Tech Fiscal Sponsorship Application

**Deadline:** Rolling (email-initiated)
**Fee:** 3-15% sliding scale
**Status:** READY TO CONTACT
**Identity:** Creative Technologist
**IRF:** IRF-INST-003

## How to Apply

1. Email **fiscalservices@aspirationtech.org** ($150 non-refundable application fee)
2. 30-minute intake interview
3. Board of Directors review and vote
4. Timeline: weeks, not months

## Model C (Pre-Approved Grant)

- Project remains a **separate legal entity** (LLC acceptable)
- **IP stays with the project** — Aspiration provides stewardship, not ownership
- Aspiration re-grants charitable contributions to the project

## What to Submit

- [ ] Project description outlining mission and values → `project-description.md`
- [ ] Budget proposal → `budget.md`
- [ ] Financial history and goals
- [ ] Demonstration of mission alignment

## Questions for Intake Call

1. Can a fiscally sponsored project maintain a parallel LLC for commercial revenue?
2. Minimum board size for Model C projects?
3. Fee tier specifics — what triggers higher rates?
4. IP licensing for permissive (MIT/Apache) vs copyleft licenses?
5. Flexibility on 1-2 year duration?
