# ORGANVM IP Policy — Open-Core Model

**Status:** DRAFT
**IRF:** IRF-INST-005

## Principle

Open-source base (through fiscal sponsor) + proprietary extensions (through LLC).
Arm's-length licensing at fair market value between entities.

## Key Rules

1. **Open-source repos stay open.** No relicensing to proprietary.
2. **Proprietary extensions built ON TOP of open-source base** — never forked from it.
3. **The LLC licenses the open-source code via its OSS license** — no special deal, same terms as anyone.
4. **Trademarks** (ORGANVM name, organ names) held by the sponsored project / future Foundation.
5. **Research papers** produced under sponsorship are public goods (open access).
6. **Commercial products** are built by the LLC, pay their own costs, generate their own revenue.

## IRS Compliance

- The nonprofit arm serves **public benefit**, not primarily the LLC
- Transactions between entities are **arm's-length, fair market value**
- Open-source licensing naturally handles this: anyone can use the code, not just the LLC
