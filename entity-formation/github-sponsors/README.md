# GitHub Sponsors Setup

**Status:** NOT STARTED
**IRF:** IRF-INST-006

## Fastest Path: Personal Account First

1. Go to GitHub Sponsors settings
2. Fill out sponsor profile (description, how funds are used)
3. Set up Stripe Connect (requires bank account + tax ID)
4. Configure tiers
5. Submit for review (days to 2-3 weeks)
6. Add FUNDING.yml to key repos

## Recommended Tiers

| Tier | Price | Perk |
|------|-------|------|
| Community | $5/mo | Name in SPONSORS.md |
| Backer | $15/mo | Logo in README |
| Bronze | $50/mo | Priority issues, monthly updates |
| Silver | $100/mo | Website logo, quarterly call |
| Gold | $500/mo | Dedicated support, roadmap input |
| Platinum | $1,000+/mo | Consulting hours, SLA |

## Fees

- Personal: **0%** (GitHub takes nothing)
- Org: up to 6% (3% processing + 3% service, eliminable via invoice billing)
